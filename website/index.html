Apa itu PAP Bot?

🔒

Privat & Aman

Foto yang kamu kirim hanya bisa dilihat oleh peserta. Semua preview dikirim secara ephemeral (hanya kamu yang bisa melihat).

🎮

4 Mode Unik

Pilih mode yang sesuai dengan kebutuhan: ROUND untuk galeri, SWAP untuk pasangan acak, CHAIN untuk berantai, atau QUOTA untuk target tertentu.

⚡

Slash Commands

Gunakan slash commands (/) yang mudah digunakan dengan autocomplete. Tidak perlu menghafal perintah rumit!

🏠

Room System

Buat room dengan kapasitas yang bisa diatur. Owner mengontrol room dengan panel yang mudah dipahami.

4 Game Modes

Pilih mode yang sesuai dengan kebutuhan server kamu

🔄

ROUND

Yang submit bisa melihat galeri peserta lain dengan pagination.

1 Owner start round
2 Semua submit foto
3 Lihat galeri (4 foto/halaman)
  • ✅ Galeri lengkap semua peserta
  • ✅ Pagination 4 foto/halaman
  • ✅ Ephemeral preview
🔀

SWAP

Peserta dipasangkan acak, lihat foto pasangan kamu.

1 Owner start round
2 Submit foto
3 Sistem acak pasangan
  • ✅ Pasangan acak otomatis
  • ✅ Hanya lihat foto pasangan
  • ✅ Seru untuk pasangan
⛓️

CHAIN

Lihat foto orang tepat sebelum kamu dalam antrian.

1 Owner start round
2 Submit foto berurutan
3 Lihat foto sebelumnya
  • ✅ Antrian berantai
  • ✅ Lihat 1 foto sebelumnya
  • ✅ Orang pertama lihat kosong
📊

QUOTA

Galeri terbuka setelah kuota minimal tercapai.

1 Set kuota (misal: 5)
2 Semua submit foto
3 Kuota tercapai = buka galeri
  • ✅ Target peserta tertentu
  • ✅ Galeri buka setelah tercapai
  • ✅ Cocok untuk event

Commands

Daftar lengkap perintah yang tersedia

Command Description Permission
/pap-help Tampilkan panduan lengkap Everyone
/pap-bootstrap Setup kategori & channel default Admin
/pap-room create [capacity] Buat room baru dengan kapasitas Everyone
/pap-room close Tutup & hapus room Owner/Admin
/pap-join Gabung ke room Everyone
/pap-leave Keluar dari room Everyone
/pap-setup mode [mode] Set mode (ROUND/SWAP/CHAIN/QUOTA) Owner/Admin
/pap-start Mulai round Owner/Admin
/pap-submit photo: Kirim foto Member
/pap-view [page] Lihat foto sesuai mode Member
/pap-close Tutup round Owner/Admin
/pap-mode Lihat status mode Everyone
/pap-stats Lihat statistik Everyone

Cara Menggunakan

1

Invite Bot

Klik tombol "Add to Discord" di atas untuk menginvite bot ke server kamu.

2

Setup Channel

Jalankan /pap-bootstrap untuk membuat kategori dan channel secara otomatis.

3

Create Room

Di channel #create-room, klik tombol "Buat Room" atau jalankan /pap-room create.

4

Setup Mode

Di room yang baru dibuat, jalankan /pap-setup mode:ROUND (atau mode lainnya).

5

Start Round

Jalankan /pap-start untuk memulai round foto sharing.

6

Share Foto

Peserta menjalankan /pap-submit photo: untuk mengirim foto.
